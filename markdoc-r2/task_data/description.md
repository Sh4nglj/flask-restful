# 任务 Prompt

为 Flask-RESTful 实现请求日志记录和审计功能，具体需求如下：
1. 在 flask_restful/utils 目录下创建 logging.py 模块，实现结构化日志记录
2. 记录每个 API 请求的详细信息：请求方法、路径、参数、响应状态码、响应时间
3. 支持多种日志格式（JSON、文本、自定义格式），便于日志分析工具处理
4. 实现日志级别控制（DEBUG、INFO、WARNING、ERROR），可配置不同资源的日志级别
5. 支持日志输出到多个目标（文件、控制台、远程服务器、数据库）
6. 实现敏感信息过滤，自动脱敏密码、token 等敏感字段
7. 添加请求 ID 追踪，便于在分布式系统中追踪请求链路
8. 实现日志轮转和归档功能，防止日志文件过大
9. 提供性能监控功能，记录慢请求（超过阈值的请求）
10. 编写使用文档和示例，说明如何配置和使用日志功能
