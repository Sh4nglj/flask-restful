# 任务 Prompt

修复 Flask-RESTful Resource 类中方法装饰器在异常情况下的处理问题，具体需求如下：
1. 修复当 method_decorators 为字典且某个 HTTP 方法不存在对应装饰器时，dispatch_request 方法可能引发的 KeyError
2. 修复当装饰器抛出异常时，异常信息丢失或无法正确传播的问题
3. 修复当 method_decorators 包含异步装饰器时，与同步方法不兼容的问题
4. 修复当装饰器修改了方法的签名（如添加参数）时，方法调用失败的问题
5. 修复当 method_decorators 为 None 或空列表时，方法调用逻辑错误
6. 修复当装饰器返回非响应对象时，响应处理逻辑不一致的问题
7. 确保装饰器的执行顺序与定义顺序一致，避免装饰器链断裂
8. 修复当装饰器内部调用 abort 时，错误处理路由无法正确识别的问题
9. 添加装饰器执行日志，便于调试装饰器相关问题
10. 编写测试用例覆盖所有装饰器异常场景，确保修复不影响正常功能

## 题目方向
运行时异常修复

## 技术栈广度
Medium: 单一框架/语言（Python/Flask）

Medium: 基础优化
